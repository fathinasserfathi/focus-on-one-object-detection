# focus-on-one-object-detection

This project demonstrates real-time object detection and tracking using Ultralytics YOLO and OpenCV. It allows you to select a detected object with a mouse click and track it continuously.

Features

Real-time object detection using YOLOv8 (yolo26l.pt model).

Object tracking using OpenCV’s CSRT tracker.

Click to select a detected object and start tracking it.

unselected objects in green.

Highlight selected object in red and remove anthor boundary boxes

Reset tracking with a key press.


Usage

Clone the repository or copy the script.

Make sure you have the YOLO model file yolo26l.pt in the same directory.

Use your mouse to click on a detected object to start tracking it.

Press r to reset tracking.

Press q to quit the application.
